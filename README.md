# MyLirc Website Files

This folder contains the HTML files for the MyLirc website hosted on GitHub Pages.

## Files

- **index.html** - Main landing page
- **privacy.html** - Privacy Policy page

## How to Upload to GitHub

1. Go to your GitHub repository: https://github.com/MyLirc/MyLirc
2. Click "Add file" â†’ "Upload files"
3. Drag and drop `index.html` and `privacy.html`
4. Click "Commit changes"
5. Wait 1-2 minutes for GitHub Pages to update

## Website URL

After uploading, your website will be available at:
- **Homepage**: https://MyLirc.github.io/MyLirc/
- **Privacy Policy**: https://MyLirc.github.io/MyLirc/privacy.html

## Customization

To customize the website:
1. Edit the HTML files in this folder
2. Upload the updated files to GitHub
3. GitHub Pages will automatically update
