1. Introducción

MyLirc es un cliente IRC (Internet Relay Chat) para Android que permite a los usuarios conectarse a servidores IRC y participar en conversaciones de chat. Esta Política de Privacidad explica cómo MyLirc recopila, usa, almacena y protege tu información.

Puntos Clave:
  • Todos los datos IRC se almacenan localmente en tu dispositivo
  • No se envían datos al desarrollador
  • Versión Free: Usa Google AdMob para publicidad (recopila ID de publicidad)
  • Versión Pro: Sin anuncios, sin seguimiento, sin servicios de terceros
  • Las contraseñas están encriptadas con AES-256

2. Recopilación de Datos

MyLirc recopila y almacena los siguientes datos localmente en tu dispositivo:

2.1 Detalles de Conexión al Servidor IRC

  • Nombre de host y puerto del servidor
  • Configuración de conexión SSL/TLS
  • Apodo y nombre de usuario del servidor
  • Nombres de canales y preferencias

2.2 Credenciales IRC (Encriptadas)

  • Contraseñas del servidor IRC (encriptadas con AES-256)
  • Credenciales de autenticación SASL (encriptadas con AES-256)
  • Contraseñas de autenticación NickServ (encriptadas con AES-256)

Seguridad: Todas las contraseñas se encriptan usando SecureStorage de Android con encriptación AES-256.

2.3 Registros de Chat (Opcional)

  • Mensajes IRC enviados y recibidos
  • Historial de conversaciones de canales
  • Historial de mensajes privados

Control del Usuario: El registro es opcional y puede deshabilitarse en Configuración.

3. Uso de Datos

Todos los datos recopilados por MyLirc se usan únicamente para:

  • Establecer conexiones IRC
  • Proporcionar funcionalidad de cliente IRC
  • Almacenar preferencias del usuario
  • Historial de chat (si está habilitado)
NO hacemos:
  • ❌ Enviar tus datos a nuestros servidores
  • ❌ Compartir tus datos con terceros
  • ❌ Usar tus datos para análisis o publicidad
  • ❌ Rastrear tu actividad o comportamiento

4. Compartir Datos

Versión Free

Google AdMob (servicio de publicidad) recopila y comparte datos:

  • ID de publicidad (para anuncios personalizados)
  • Información del dispositivo (modelo, versión de SO)
  • Dirección IP (ubicación aproximada)
  • Datos de uso de la app (para segmentación de anuncios)

Nosotros (el desarrollador) NO accedemos ni almacenamos tu ID de publicidad.

Versión Pro

MyLirc Pro NO comparte ningún dato de usuario con terceros.

5. Servicios de Terceros

Versión Free

Google AdMob:

  • Propósito: Mostrar anuncios para apoyar el desarrollo gratuito de la app
  • Política de Privacidad: https://policies.google.com/privacy
  • Opt-out: Los usuarios pueden restablecer el ID de publicidad en Configuración de Android > Google > Anuncios

Versión Pro

La versión Pro NO usa ningún servicio de terceros para publicidad, análisis o seguimiento.

6. Seguridad de Datos

  • Encriptación AES-256 para todas las contraseñas
  • Almacenamiento local seguro usando SecureStorage de Android
  • Soporte SSL/TLS para conexiones IRC
  • Sin transmisión de datos al desarrollador

7. Tus Derechos

Puedes:

  • ✅ Ver todos tus datos en Configuración
  • ✅ Eliminar todos los datos desinstalando la app
  • ✅ Deshabilitar el registro en Configuración
  • ✅ Eliminar registros de chat en cualquier momento

8. Contacto

¿Preguntas? Contáctanos en: c4lm40s@gmail.com

1. Introduction

MyLirc is an IRC (Internet Relay Chat) client for Android that allows users to connect to IRC servers and participate in chat conversations. This Privacy Policy explains how MyLirc collects, uses, stores, and protects your information.

Key Points:
  • All IRC data is stored locally on your device
  • No data is sent to us (the developer)
  • Free version: Uses Google AdMob for advertising (collects Advertising ID)
  • Pro version: No ads, no tracking, no third-party services
  • Passwords are encrypted using AES-256 encryption

2. Data Collection

MyLirc collects and stores the following data locally on your device:

2.1 IRC Server Connection Details

  • Server hostname and port
  • SSL/TLS connection settings
  • Server nickname and username
  • Channel names and preferences

2.2 IRC Credentials (Encrypted)

  • IRC server passwords (encrypted using AES-256)
  • SASL authentication credentials (encrypted using AES-256)
  • NickServ authentication passwords (encrypted using AES-256)

Security: All passwords are encrypted using Android's SecureStorage with AES-256 encryption.

2.3 Chat Logs (Optional)

  • IRC messages sent and received
  • Channel conversation history
  • Private message history

User Control: Logging is optional and can be disabled in Settings.

3. Data Usage

All data collected by MyLirc is used solely for:

  • Establishing IRC connections
  • Providing IRC client functionality
  • Storing user preferences
  • Chat history (if enabled)
We do NOT:
  • ❌ Send your data to our servers
  • ❌ Share your data with third parties
  • ❌ Use your data for analytics or advertising
  • ❌ Track your activity or behavior

4. Data Sharing

Free Version

Google AdMob (advertising service) collects and shares data:

  • Advertising ID (for personalized ads)
  • Device information (model, OS version)
  • IP address (approximate location)
  • App usage data (for ad targeting)

We (the developer) do NOT access or store your Advertising ID.

Pro Version

MyLirc Pro does NOT share any user data with third parties.

5. Third-Party Services

Free Version

Google AdMob:

  • Purpose: Display advertisements to support free app development
  • Privacy Policy: https://policies.google.com/privacy
  • Opt-out: Users can reset Advertising ID in Android Settings > Google > Ads

Pro Version

The Pro version does NOT use any third-party services for advertising, analytics, or tracking.

6. Data Security

  • AES-256 encryption for all passwords
  • Secure local storage using Android's SecureStorage
  • SSL/TLS support for IRC connections
  • No data transmission to the developer

7. Your Rights

You can:

  • ✅ View all your data in Settings
  • ✅ Delete all data by uninstalling the app
  • ✅ Disable logging in Settings
  • ✅ Delete chat logs at any time

Questions? Contact us at: c4lm40s@gmail.com
