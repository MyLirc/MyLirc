Privacy Policy for MyLirc IRC Client

Last Updated: December 25, 2024

Effective Date: December 25, 2024

Key Points:
  • ✅ All IRC data stored locally on your device
  • ✅ No data sent to us (the developer)
  • ⚠️ Free version: Uses Google AdMob (collects Advertising ID)
  • ✅ Pro version: No ads, no tracking, no third parties
  • ✅ Passwords encrypted using AES-256

1. Introduction

MyLirc ("we", "our", or "the app") is an IRC (Internet Relay Chat) client for Android that allows users to connect to IRC servers and participate in chat conversations. This Privacy Policy explains how MyLirc collects, uses, stores, and protects your information.

2. Data Collection

MyLirc collects and stores the following data locally on your device:

2.1 IRC Server Connection Details

  • Server hostname and port
  • SSL/TLS connection settings
  • Server nickname and username
  • Channel names and preferences

2.2 IRC Credentials (Encrypted)

  • IRC server passwords (encrypted using AES-256)
  • SASL authentication credentials (encrypted using AES-256)
  • NickServ authentication passwords (encrypted using AES-256)

2.3 Chat Logs (Optional)

  • IRC messages sent and received
  • Channel conversation history
  • Private message history

User Control: Logging is optional and can be disabled in Settings. Users can delete chat logs at any time.

3. Data Usage

All data collected by MyLirc is used solely for the following purposes:

  1. Establishing IRC Connections: Server details and credentials are used to connect to IRC servers you configure.
  2. Providing IRC Functionality: Messages, commands, and user lists are processed to provide IRC client features.
  3. Storing User Preferences: Settings and preferences are saved to personalize your experience.
  4. Chat History: If logging is enabled, messages are stored locally for your reference.
We do NOT:
  • ❌ Send your data to our servers
  • ❌ Share your data with third parties
  • ❌ Use your data for analytics or advertising
  • ❌ Track your activity or behavior

4. Data Storage and Security

4.1 Local Storage

All data is stored locally on your Android device. No cloud storage or remote servers are used.

4.2 Encryption

  • Passwords: Encrypted using AES-256 encryption via Android's KeyStore system
  • In Transit: IRC connections support SSL/TLS encryption when configured
  • At Rest: Passwords are encrypted on disk; other data is stored in app-private storage

4.3 Network Security

  • The app enforces HTTPS-only connections for secure communication
  • Cleartext (HTTP) traffic is blocked by default
  • SSL/TLS certificate validation is enforced

5. Data Sharing

Free Version

Google AdMob (advertising service) collects and shares data:

  • ✅ Advertising ID: Shared with ad networks for personalized ads
  • ✅ Device Info: Model, OS version, screen size
  • ✅ IP Address: For approximate location-based ads
  • ✅ App Usage: For ad targeting and analytics

We (the developer) do NOT:

  • ❌ Access or store your Advertising ID
  • ❌ Share your IRC data with anyone
  • ❌ Sell your personal information

Pro Version

MyLirc Pro does NOT share any user data with third parties.

The only external connections made by the app are to IRC servers you explicitly configure. No analytics, no advertising, no tracking.

6. Data Retention and Deletion

6.1 Data Deletion

You can delete your data in the following ways:

  • Delete Chat Logs: Settings > Logging > Clear Chat History
  • Delete Server Configurations: Long-press a server > Delete
  • Delete All App Data: Android Settings > Apps > MyLirc > Storage > Clear Data
  • Complete Removal: Uninstalling the app will permanently delete all data

7. Third-Party Services

Free Version

The Free version uses the following third-party service:

Google AdMob (Advertising)

  • Purpose: Display advertisements to support free app development
  • Data Collected by AdMob:
    • Advertising ID (for personalized ads)
    • Device information (model, OS version)
    • IP address (approximate location)
    • App usage data (for ad targeting)
  • Privacy Policy: https://policies.google.com/privacy
  • Opt-out: Users can reset Advertising ID in Android Settings > Google > Ads
Important: AdMob may share data with advertising partners. We do not control this data collection.

Pro Version

The Pro version does NOT use any third-party services for:

  • ✅ No advertising (no AdMob)
  • ✅ No analytics (no Google Analytics, Firebase)
  • ✅ No tracking (no data collection)
  • ✅ No cloud storage (no Google Drive, Dropbox)

Development Libraries (Both Versions)

Both versions use these standard development libraries that do not collect user data:

  • AndroidX libraries (Google's official Android libraries)
  • Hilt/Dagger (dependency injection framework)
  • Material Components (Google's UI library)
  • Kotlin Coroutines (concurrency framework)
